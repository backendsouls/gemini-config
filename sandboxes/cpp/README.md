# C++ Sandbox

This sandbox is optimized for C/C++ development and includes essential build tools.

## Included
- **Tools**: `build-essential`, `cmake`, `gdb` (via Homebrew/Base)
- **Base Tools**: `jq`, `fzf`, `rg`, `fd`, `bat`

## Usage
```bash
docker build -t gemini-sandbox-cpp .
docker run -it gemini-sandbox-cpp
```
