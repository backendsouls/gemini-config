# Ruby Sandbox

This sandbox is optimized for Ruby development.

## Included
- **Language**: Ruby (via Homebrew)
- **Scripting**: Python 3.11 (for Gemini usage)
- **Base Tools**: `jq`, `fzf`, `rg`, `fd`, `bat`

## Usage
```bash
docker build -t gemini-sandbox-ruby .
docker run -it gemini-sandbox-ruby
```
