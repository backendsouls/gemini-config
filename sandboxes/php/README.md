# PHP Sandbox

This sandbox is optimized for PHP development.

## Included
- **Language**: PHP (via Homebrew)
- **Scripting**: Python 3.11 (for Gemini usage)
- **Base Tools**: `jq`, `fzf`, `rg`, `fd`, `bat`

## Usage
```bash
docker build -t gemini-sandbox-php .
docker run -it gemini-sandbox-php
```
