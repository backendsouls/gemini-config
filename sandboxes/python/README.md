# Python Sandbox

This sandbox is optimized for Python development and includes Gemini-ready scripting capabilities.

## Included
- **Language**: Python 3.11 (via Homebrew)
- **Base Tools**: `jq`, `fzf`, `rg`, `fd`, `bat`

## Usage
```bash
docker build -t gemini-sandbox-python .
docker run -it gemini-sandbox-python
```
