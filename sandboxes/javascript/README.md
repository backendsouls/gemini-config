# JavaScript Sandbox

This sandbox is optimized for JavaScript/Node.js development.

## Included
- **Language**: Node.js (latest via Homebrew)
- **Scripting**: Python 3.11 (for Gemini usage)
- **Base Tools**: `jq`, `fzf`, `rg`, `fd`, `bat`

## Usage
```bash
docker build -t gemini-sandbox-js .
docker run -it gemini-sandbox-js
```
